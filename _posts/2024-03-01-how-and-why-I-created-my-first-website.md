## Options
There are too many ways to build a website.
### But mainly there are two types of websites.
 - Dynamic
 - Static

### Dynamic
Web pages that change according to the user who was using it based on their credentials are called `**dynamic**` websites. Building a dynamic website is time-consuming and needs a lot of afford to maintain.

### Static
When a Website appears the same for every user who accesses it, is a **`static`** website. This type of website is easy to build and maintain.

![lets-keep-it-clean-maintain.gif](https://i.postimg.cc/cC2r0TN2/lets-keep-it-clean-maintain.gif){: width='400'}
### Conclusion
Our requirement is to show blog posts. And on every new post, the website should get updated. Based on requirements we're going to build a **static** site. We do care about our time and energy. so we're going to build this using some framework/tools to save our time and energy.
